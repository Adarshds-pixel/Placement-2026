------------------------------------------------------------
PROBLEM STATEMENT
------------------------------------------------------------

Given an integer array nums,

Return the Pivot Index.

Pivot Index is the index where

Sum of all elements on the LEFT

=

Sum of all elements on the RIGHT.

The current element is NOT included in either sum.

If no Pivot Index exists,

return -1.

------------------------------------------------------------
EXAMPLE
------------------------------------------------------------

Input

nums = [1,7,3,6,5,6]

Output

3

Explanation

Left Side

1 + 7 + 3 = 11

Right Side

5 + 6 = 11

Since both are equal,

Pivot Index = 3

------------------------------------------------------------
IMPORTANT OBSERVATION
------------------------------------------------------------

At Pivot Index,

Current Element

does NOT belong to

Left Side

or

Right Side.

Example

[1,7,3,6,5,6]
       ↑

Left Sum

1+7+3

Right Sum

5+6

Current element (6)

is ignored.

------------------------------------------------------------
BRUTE FORCE APPROACH
------------------------------------------------------------

Idea

Check every index.

For every index,

calculate

Left Sum

and

Right Sum

separately.

If both are equal,

return that index.

------------------------------------------------------------
BRUTE FORCE DRY RUN
------------------------------------------------------------

nums = [1,7,3,6,5,6]

i = 0

Left Sum = 0

Right Sum = 7+3+6+5+6 = 27

Not Equal

--------------------------------

i = 1

Left Sum = 1

Right Sum = 3+6+5+6 = 20

Not Equal

--------------------------------

i = 2

Left Sum = 1+7 = 8

Right Sum = 6+5+6 = 17

Not Equal

--------------------------------

i = 3

Left Sum = 1+7+3 = 11

Right Sum = 5+6 = 11

Equal

Return 3

------------------------------------------------------------
BRUTE FORCE CODE
------------------------------------------------------------

class Solution:
    def pivotIndex(self, nums: List[int]) -> int:

        for i in range(len(nums)):

            left_sum = sum(nums[:i])

            right_sum = sum(nums[i+1:])

            if left_sum == right_sum:
                return i

        return -1

OPTIMAL CODE
------------------------------------------------------------

class Solution:
    def pivotIndex(self, nums: List[int]) -> int:

        total_sum = sum(nums)

        left_sum = 0

        for i in range(len(nums)):

            right_sum = total_sum - left_sum - nums[i]

            if left_sum == right_sum:
                return i

            left_sum += nums[i]

        return -1
