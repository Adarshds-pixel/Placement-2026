------------------------------------------------------------
PROBLEM STATEMENT
------------------------------------------------------------

Given an integer array nums,

Return an array where

runningSum[i] = nums[0] + nums[1] + ... + nums[i]

In simple words,

Each element should contain the sum of itself and all previous elements.

------------------------------------------------------------
EXAMPLE
------------------------------------------------------------

Input:

nums = [1,2,3,4]

Output:

[1,3,6,10]

Explanation:

Index 0 → 1

Index 1 → 1+2 = 3

Index 2 → 1+2+3 = 6

Index 3 → 1+2+3+4 = 10

------------------------------------------------------------
OBSERVATION
------------------------------------------------------------

Every answer depends on the previous elements.

Current Answer

=

Previous Sum

+

Current Element

This is our first Prefix Sum problem.

------------------------------------------------------------
BRUTE FORCE APPROACH
------------------------------------------------------------

Idea:

For every index,

start from index 0

calculate the sum again and again.

Example:

nums = [1,2,3,4]

For index 0

sum = 1

For index 1

sum = 1+2 = 3

For index 2

sum = 1+2+3 = 6

For index 3

sum = 1+2+3+4 = 10

Notice

For every index

we are starting from 0 again.

Lots of repeated work.

------------------------------------------------------------
BRUTE FORCE CODE
------------------------------------------------------------

class Solution:
    def runningSum(self, nums: List[int]) -> List[int]:

        result = []

        for i in range(len(nums)):

            total = 0

            for j in range(i + 1):
                total += nums[j]

            result.append(total)

        return result

OPTIMAL CODE
------------------------------------------------------------

class Solution:
    def runningSum(self, nums: List[int]) -> List[int]:

        i = 1

        while i < len(nums):

            nums[i] += nums[i-1]

            i += 1

        return nums
