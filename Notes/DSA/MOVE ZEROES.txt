2. Move Zeroes (LeetCode 283)
Problem

Move all zeros to the end while maintaining order of non-zero elements.

Example:

Input:  [0,1,0,3,12]

Output: [1,3,12,0,0]
Brute Force

Create new array.

Store non-zero elements.

Append zeros at end.

Complexity
Time: O(N)
Space: O(N)
Optimal Solution
Idea

Use a pointer called:

right

which stores the next position for non-zero element.

Code
right = 0

for i in range(len(nums)):
    if nums[i] != 0:
        nums[right] = nums[i]
        right += 1

for i in range(right, len(nums)):
    nums[i] = 0
Dry Run
[0,1,0,3,12]

Store non-zeroes:

[1,3,12,3,12]

Fill remaining with zeros:

[1,3,12,0,0]
Complexity
Time: O(N)
Space: O(1)
Pattern
Two Pointers
