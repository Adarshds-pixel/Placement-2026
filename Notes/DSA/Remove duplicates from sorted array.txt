Problem
-------
Remove duplicates from a sorted array.

Modify the array in-place.

Return the number of unique elements.

Pattern
-------
Same Direction (Fast & Slow Pointer)

Idea
----
Fast explores every element.

Slow keeps track of the last unique element.

Whenever Fast finds a new value,
move Slow and copy Fast's value.

Rule
----
Fast Explores.

Slow Builds.

Algorithm
---------
1. slow = 0

2. Fast starts from index 1.

3. If nums[fast] != nums[slow]

       slow++

       nums[slow] = nums[fast]

4. Return slow + 1

Why return slow + 1 ?
---------------------

slow stores INDEX.

Question asks for COUNT.

Index starts from 0.

Count starts from 1.

Count = Index + 1

Template
--------

slow = 0

for fast in range(1, len(nums)):

    if nums[fast] != nums[slow]:

        slow += 1

        nums[slow] = nums[fast]

return slow + 1

Time Complexity
---------------
O(n)

Space Complexity
----------------
O(1)

Interview Tip
-------------
Whenever you see

• In-place modification
• Remove duplicates
• Remove elements
• Compress array

Think

Fast & Slow Pointer.
