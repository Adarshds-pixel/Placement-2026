WHAT IS HTML?
-------------

HTML stands for

HyperText Markup Language

HTML is the standard markup language used to create the structure of web pages.

It tells the browser WHAT should be displayed.

HTML is NOT a programming language.

It is a Markup Language.

-----------------------------------------------------------

WHY DO WE NEED HTML?
--------------------

HTML is used to:

• Create web pages
• Structure website content
• Display text, images, videos and links
• Build forms
• Create tables
• Connect CSS and JavaScript

Without HTML,

There is no webpage.

-----------------------------------------------------------

WHAT IS A MARKUP LANGUAGE?
--------------------------

A Markup Language uses tags to define elements.

Example

<h1>Hello World</h1>

Here

<h1>

defines a Heading.

-----------------------------------------------------------

WHAT IS HYPERTEXT?
------------------

HyperText means text containing links.

Example

<a href="https://google.com">
Google
</a>

Clicking this link takes us to another webpage.

BASIC HTML STRUCTURE
--------------------

<!DOCTYPE html>
<html>

<head>

<title>My Website</title>

</head>

--------------------------------

<title>

BEST PRACTICES
--------------

• Use semantic tags.

• Always close tags.

• Use meaningful names.

• Write proper indentation.

• Add alt attribute for images.

• Use labels in forms.

• Keep code readable.

-----------------------------------------------------------

COMMON MISTAKES
---------------

❌ Forgetting closing tags.

❌ Using too many <br> tags.

❌ Not adding alt attribute.

❌ Wrong nesting of elements.

❌ Using <b> instead of <strong> for important text.
