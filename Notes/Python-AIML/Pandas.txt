# What is Pandas?

Pandas is an open-source Python library used for data manipulation and analysis.

It provides fast and flexible data structures that make working with structured data easy.

### Applications

* Data Cleaning
* Data Analysis
* Data Manipulation
* Machine Learning Preprocessing
* Data Visualization Preparation

---

# Importing Pandas

```python
import pandas as pd
```

`pd` is the standard alias used for Pandas.

---

# Pandas Data Structures

Pandas mainly provides two data structures:

## 1. Series

A one-dimensional labeled array.

Example:

```python
import pandas as pd

s = pd.Series([10, 20, 30, 40])

print(s)
```

Output:

```
0    10
1    20
2    30
3    40
dtype: int64
```

### Features

* One-dimensional
* Stores single column of data
* Has index labels

---

## Creating Series

### From List

```python
numbers = pd.Series([5, 10, 15])
```

### With Custom Index

```python
numbers = pd.Series(
    [5,10,15],
    index=["A","B","C"]
)
```

Output

```
A    5
B   10
C   15
```

---

# DataFrame

A DataFrame is a two-dimensional table consisting of rows and columns.

It is the most commonly used Pandas data structure.

Example

```python
import pandas as pd

data = {
    "Name":["Alice","Bob"],
    "Age":[21,22]
}

df = pd.DataFrame(data)

print(df)
```

Output

```
   Name   Age
0 Alice   21
1 Bob     22
```

---

# Creating DataFrame

### Using Dictionary

```python
data = {
    "Name":["John","Alice"],
    "Age":[20,21],
    "City":["Delhi","Mumbai"]
}

df = pd.DataFrame(data)
```

---

### Using List of Lists

```python
data = [
    ["John",20],
    ["Alice",21]
]

df = pd.DataFrame(
    data,
    columns=["Name","Age"]
)
```

# Interview Questions

### What is Pandas?

A Python library used for data manipulation and analysis.

### Difference between Series and DataFrame?

Series is one-dimensional, while DataFrame is two-dimensional.

### Difference between loc() and iloc()?

* loc() uses labels.
* iloc() uses integer positions.

### Difference between head() and tail()?

* head() returns first rows.
* tail() returns last rows.

### Difference between dropna() and fillna()?

* dropna() removes missing values.
* fillna() replaces missing values.
