WHAT IS NUMPY?
--------------

NumPy (Numerical Python) is a Python library used for:

• Numerical Computing
• Mathematical Operations
• Matrix Operations
• Scientific Computing
• AI & Machine Learning
• Data Analysis

It provides a powerful N-dimensional array (ndarray) which is much faster than Python Lists.

-----------------------------------------------------------

WHY DO WE NEED NUMPY?
---------------------

Python Lists have limitations for mathematical computations.

Problems with Python Lists:

• Slower for numerical operations
• Higher memory overhead
• No element-wise arithmetic
• Requires loops for calculations

NumPy solves these problems by:

• Faster execution
• Better memory management
• Vectorized operations
• Broadcasting
• Optimized compiled C implementation

-----------------------------------------------------------

WHY IS NUMPY FASTER?
--------------------

Reasons:

1. Continuous Memory Allocation

Python List

Object -> Pointer -> Value

NumPy Array

Value Value Value Value

Stored continuously in memory.

--------------------------------

2. Optimized C Implementation

NumPy performs computations using optimized compiled C code.

--------------------------------

3. Vectorization

Operations are applied on the entire array at once.

No explicit Python loops required.

-----------------------------------------------------------

VECTORIZATION
-------------

Definition

Performing mathematical operations on an entire array at once without writing explicit Python loops.

Example

arr = np.array([2,3,4])

arr * 10

Output

[20 30 40]

Advantages

• Faster
• Cleaner code
• Uses optimized compiled code
• Used heavily in AI/ML

-----------------------------------------------------------

BROADCASTING
------------

Definition

Broadcasting allows NumPy to perform operations between arrays of different shapes by automatically expanding the smaller array without duplicating data.

Example

arr = np.array([2,3,4])

arr + 10

Output

[12 13 14]

NumPy automatically treats

10

as

[10 10 10]

without actually creating that array.

-----------------------------------------------------------

RESHAPE
-------

Changes the shape of an array.

Example

arr = np.array([1,2,3,4,5,6])

arr.reshape(2,3)

Output

[[1 2 3]
 [4 5 6]]

Rule

Total number of elements must remain same.

2 × 3 = 6

--------------------------------

Using -1

arr.reshape(-1,3)

NumPy automatically calculates required rows.

COPY VS ASSIGNMENT
------------------

Assignment

b = a

Both variables point to same memory.

Changing one changes the other.

--------------------------------

Copy

b = a.copy()

Creates a new independent array.

Changes do not affect original array.

INTERVIEW QUESTIONS
-------------------

Q. Why is NumPy faster than Python Lists?

Answer:

Because NumPy stores data in continuous memory and performs operations using optimized compiled C code with vectorization.

--------------------------------

Q. What is Vectorization?

Performing operations on an entire array without writing Python loops.

--------------------------------

Q. What is Broadcasting?

Automatically expanding arrays of compatible shapes for arithmetic operations without copying data.

--------------------------------

Q. Difference between Python List and NumPy Array?

Python List

• Slower
• More memory
• No element-wise arithmetic
• General-purpose

NumPy Array

• Faster
• Memory efficient
• Element-wise operations
• Optimized for numerical computing
