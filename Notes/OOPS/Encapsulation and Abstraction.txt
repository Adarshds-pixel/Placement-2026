3. Constructor
Definition

Special method executed automatically when object is created.

Syntax
class Student:

    def __init__(self,name):
        self.name = name
Purpose

Initialize object attributes.

4. self Keyword
Purpose

Refers to current object.

Example
self.name = name

means:

Store name inside current object.
Why not
name = name

?

Because nothing gets stored inside object.

Rule
self.attribute

→ Attribute belongs to object

Goal

Maintainability

Example

Interview Definition
