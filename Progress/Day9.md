# 💡 Key Learnings

- Recursion solves smaller versions of the same problem.
- Merge Sort divides first and sorts while merging.
- Quick Sort places the pivot in its correct position first, then recursively sorts both sides.
- Merge Sort is stable and guarantees O(n log n).
- Quick Sort is generally faster in practice but can degrade to O(n²).
- Dutch National Flag Algorithm sorts 0s, 1s, and 2s using three pointers in a single traversal.
